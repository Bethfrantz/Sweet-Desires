🍰 Sweet Desires — Handcrafted Bakery Website
Sweet Desires is a warm and cozy multi‑page bakery website built using semantic HTML, modern CSS, and light JavaScript. The site showcases a handcrafted brand identity with original photography, responsive layouts, and a clean, inviting design.

This project was originally created for an academic assignment and has since been upgraded to reflect professional, real‑world web development practices.

✨ Features
Multi‑page website with consistent branding

Warm, cozy visual theme using all original images

Responsive layout using modern CSS Grid and Flexbox

Hero banners for each page

Clean, semantic HTML structure

Accessible navigation and form labels

Upgraded gallery and menu layouts (no tables)

Organized file structure for scalability

🧁 Pages Included
Home — Welcome message, hero banner, and quick links

Weddings — Cake options, flavors, fillings, and customizations

Special Events — Image gallery of birthday, baby shower, and celebration cakes

Desserts — Modern menu layout with pricing and categories

Contact — Clean, accessible contact form

🎨 Tech Stack
HTML5

CSS3 (Grid, Flexbox, responsive design)

JavaScript (form handling)

Custom photography (all images created by me)
